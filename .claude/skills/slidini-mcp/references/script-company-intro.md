# 会社紹介向け台本ガイド

## ターゲット視聴者

求職者、取引先候補、投資家、一般消費者など、会社について知りたい幅広い層がターゲット。ユーザーが「採用向け」「IR向け」「取引先向け」など具体的な用途を指定した場合は、そのペルソナに焦点を合わせる。指定がなければ汎用的な会社紹介として構成する。

## トーン

「です・ます」調で、誠実かつ温かみのある語り口を基本とする。自慢げにならず、等身大の姿勢で会社の魅力を伝える。「この会社で働いている人が、友人に自社を紹介している」ような自然さを目指す。

## フックの切り口

会社紹介の視聴者は「この会社はどんな会社なのか」をざっくり知りたくて見始める。以下のようなパターンが効果的：

- **ミッション先行型**: 「私たちは○○な世界を目指しています」
- **数字インパクト型**: 「創業○年、○カ国で展開、○万人が利用」
- **ストーリー型**: 「すべては○○という一つの想いから始まりました」
- **問いかけ型**: 「○○な社会を、一緒に作りませんか？」（採用向け）

## 本題の構成パターン

会社紹介では「何をしている会社か → なぜやっているか → どうやっているか → これからどうなるか」の流れが王道：

### 汎用パターン
1. 会社概要（事業内容・規模・拠点）
2. ミッション・ビジョン・バリュー
3. 主要な事業・サービス
4. 実績・数字で見る成長
5. カルチャー・働き方
6. 今後の展望

### 採用向けパターン
1. 会社が解決している課題
2. 事業の面白さ・社会的意義
3. チーム・カルチャー紹介
4. キャリアパス・成長環境
5. 福利厚生・働き方
6. 募集ポジション・応募方法

### IR・投資家向けパターン
1. 事業概要・市場ポジション
2. ビジネスモデル
3. 財務ハイライト
4. 成長戦略
5. 競争優位性
6. 経営チーム

## ビジュアル要素

共通のビジュアル要素に加え、以下を積極的に活用する：

### タイムライン

創業から現在までの沿革を時系列で見せる。主要なマイルストーン（創業、資金調達、上場、海外展開など）をハイライトする。

### 数字で見る会社

重要なKPIを大きな数字で見せるスライドは印象に残りやすい。
例：「従業員数 500名 / 導入企業 2,000社 / NPS 72」

### 組織図・チーム構成

会社の規模感やチーム構成を視覚的に示す。採用向けの場合は特に、配属先のチーム体制を見せると応募のイメージが湧きやすい。

### オフィス・働く環境の写真指示

「オフィスの写真を挿入」「リモートワークの様子」など、写真素材の挿入指示を具体的に書く。

### ミッション・ビジョンの大きな文字表示

会社の理念やバリューは、シンプルに大きな文字で表示するのが最も効果的。装飾はミニマルに。

## まとめでの行動喚起

用途によって変わる：

- **採用向け**: 採用ページへのアクセス、カジュアル面談の申し込み
- **取引先向け**: お問い合わせ、資料請求
- **IR向け**: IRページへのアクセス、決算資料の閲覧
- **汎用**: 公式サイト訪問、SNSフォロー

## ファクトチェックの重点項目

会社紹介は「公式情報」としての性格が強いため、正確性は特に重要：

- 設立年、従業員数、拠点数、売上などの企業データが最新か
- 受賞歴、認証、メディア掲載の事実確認
- 経営陣の肩書き・氏名が正しいか
- 取引先・パートナー企業名の掲載許可があるか（ユーザーに確認を促す）
- 財務データはIR資料や有価証券報告書と一致しているか

## 留意事項

- 抽象的なビジョンだけでは伝わらない。具体的なエピソードや数字を必ずセットにする
- 社員の声や社内の雰囲気を伝えるパートは、採用向けに限らず効果的。「人」が見えると企業への親近感が湧く
- 競合他社との差別化ポイントは、直接的な比較ではなく自社の独自性として語るほうが品がよい
- 企業データ（従業員数、売上など）は変動するため、動画内では「○○年○月時点」と明記するようユーザーに提案する
