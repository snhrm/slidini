# 営業資料向け台本ガイド

## ターゲット視聴者

見込み顧客、既存顧客、または社内の営業チームを主なターゲットとする。視聴者は必ずしも技術に詳しくないため、専門用語は避けるか、使う場合は平易な言い換えを添える。「この動画を見た後に行動を起こしたくなる」ことがゴール。

## トーン

「です・ます」調で、信頼感と親しみやすさのバランスを取る。「頼れるコンサルタントが提案している」ような雰囲気を目指す。過度な煽りや誇張は避け、データと事例で語る誠実なスタイルが効果的。

## フックの切り口

営業対象の視聴者は「自社の課題が解決できそう」という期待で見始める。以下のようなパターンが効果的：

- **課題共感型**: 「○○でお悩みではありませんか？」「多くの企業が○○で苦労しています」
- **数字インパクト型**: 「導入企業の○%が□□を達成」「平均○○%のコスト削減に成功」
- **事例先行型**: 「A社は○○でこの問題を解決しました」
- **市場トレンド型**: 「○○業界では今、□□が急速に広まっています」

## 本題の構成パターン

営業資料では「課題 → 解決策 → 証拠」の流れが鉄板：

- **課題の深掘り → ソリューション提示 → 導入事例 → ROI**: 提案型プレゼンに最適
- **市場動向 → 競合比較 → 自社の強み → 次のステップ**: 差別化訴求に最適
- **Before/After → 仕組み解説 → 料金プラン → Q&A**: サービス説明会に最適

### 構成上の注意点

- 最初の3スライドで「この動画は自分に関係ある」と思わせる。業界特有の課題や数字を冒頭に置く
- 機能の羅列ではなく、「その機能で何が解決されるか」をセットで伝える
- 競合との比較は公平に行い、自社の弱みも正直に触れたうえで強みを際立たせるほうが信頼を得やすい

## ビジュアル要素

共通のビジュアル要素に加え、以下を積極的に活用する：

### ROI・数値グラフ

導入効果を視覚化するグラフは営業資料の要。棒グラフ・折れ線グラフの指示を具体的に書く。

例：「棒グラフ: 導入前の月間コスト vs 導入後の月間コスト。導入後が40%削減されていることを強調」

### 顧客の声・引用

実際の導入企業からのコメントや評価を引用形式で表示する。社名・担当者名（許可済みの場合）を添えると信頼性が増す。

### Before/After比較

課題がある状態（Before）とソリューション導入後（After）を並べて見せるのが効果的。数値の変化を含めるとインパクトが大きい。

### プロセスフロー

導入ステップや利用フローを視覚的に示す。「3ステップで導入完了」のようなシンプルさを伝える。

## まとめでの行動喚起

- 無料トライアル・デモの申し込み
- 資料請求・問い合わせ
- 個別相談の予約
- 概要欄のリンクから詳細を確認

## ファクトチェックの重点項目

営業資料では誇張された数字が信頼を一気に損なう。特に以下を重点的に検証する：

- 導入効果の数値（○%削減、○倍向上など）の出典
- 導入企業数・利用者数が最新か
- 競合との比較が公平な条件で行われているか
- 料金・プランの情報が最新か
- 受賞歴やメディア掲載の事実確認

## 留意事項

- 「売り込み感」が強すぎると逆効果。事実とデータで語り、視聴者自身に判断してもらうスタンスが結果的に成約率を上げる
- 視聴者が社内稟議に使えるような客観的な情報を盛り込むと、動画の再利用性が高まる
- 料金やプランの詳細は変更が多いため、動画内では大枠を示し、詳細は概要欄のリンクに誘導するほうが運用しやすい
