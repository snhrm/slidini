# YouTube 概要欄生成スキル

slidini スライド動画の台本（`*-script.md`）と `.slide.json` から、YouTube 概要欄テキストを生成する。

## 入力

以下のいずれか（または両方）を読み取る:

1. **台本ファイル** (`*-script.md`): スライドごとのタイトル・ナレーションテキスト
2. **`.slide.json`**: `playback.slides[]` の `duration`（秒）からタイムスタンプを算出

プロジェクトパスが指定された場合は `projects/<name>/` 配下を自動探索する。

## 生成手順

### 1. タイムスタンプの算出

`.slide.json` の `playback.slides[]` から各スライドの開始時刻を累積計算する:

```
slide 0: 0:00
slide 1: 0:00 + slide0.duration
slide 2: 0:00 + slide0.duration + slide1.duration
...
```

`MM:SS` 形式にフォーマットする（1時間超の場合は `H:MM:SS`）。

### 2. 目次項目の決定

台本ファイルの `## スライド N: タイトル` からセクション名を抽出する。
以下のルールで目次を構成する:

- カバースライド（スライド1）は動画タイトルそのものなので、短い見出しにする
- 連続する関連スライドは1つの目次項目にまとめてよい（詳細スライドは親トピックに含める）
- 絵文字は使わない
- 目次項目数の目安: 動画全体で15〜30項目

### 3. 概要欄の構成

以下のセクションを順番に出力する:

```
{導入文 2〜4行}

━━━━━━━━━━━━━━━━━━━━━━
目次
━━━━━━━━━━━━━━━━━━━━━━
{タイムスタンプ付き目次}

━━━━━━━━━━━━━━━━━━━━━━
出典・参考リンク
━━━━━━━━━━━━━━━━━━━━━━
{出典セクション}

━━━━━━━━━━━━━━━━━━━━━━
クレジット
━━━━━━━━━━━━━━━━━━━━━━
音声: VOICEVOX:春日部つむぎ
https://voicevox.hiroshiba.jp/

{ハッシュタグ行}
```

## 各セクションの書き方

### 導入文

- 動画の内容を2〜4行で簡潔にまとめる
- 1行目: 何についての動画か（テーマ・対象期間など）
- 2行目以降: 見どころ・ハイライトを具体的に（数字やキーワードを含める）
- 煽りすぎない、事実ベースのトーン

### タイムスタンプ目次

```
00:00 動画タイトル or 導入
00:13 ハイライト
00:39 はじめに
00:57 セクション名
01:16 注目トピック
...
```

- 先頭は必ず `00:00` から始める
- インデントなし、フラットなリスト
- 絵文字は使わない（目次、セクション見出し含め全体で不使用）

### 出典・参考リンク

- 動画内で言及した公式サイト・ドキュメント・論文・記事へのリンク
- カテゴリごとに `▼ カテゴリ名` で区切る
- `表示名: URL` の形式

### ハッシュタグ

- 動画のテーマに関連するハッシュタグを10〜20個
- 日本語・英語を混ぜる
- 汎用的なもの（`#AI`, `#エンジニア`）と具体的なもの（`#ClaudeCode`, `#VSCode`）をバランスよく
- 1行にスペース区切りで並べる

## クレジットのデフォルト

音声合成に VOICEVOX を使用している場合（`.slide.json` の `playback` にナレーション音声がある場合）:

```
音声: VOICEVOX:春日部つむぎ
https://voicevox.hiroshiba.jp/
```

ユーザーが別のキャラクターや音声合成エンジンを指定した場合はそちらに従う。

## 文字数制限

YouTube概要欄の上限は5,000文字。生成後に文字数を確認し、超過する場合は以下の優先度で削る:

1. 出典リンクのうち重要度が低いものを削除
2. 目次項目を統合して数を減らす
3. 導入文を短縮

最終出力時に文字数を表示する（例: `【3,245文字 / 5,000文字】`）。

## 出力

生成した概要欄テキストを `projects/<name>/<name>-description.md` に保存し、内容をそのまま表示する。ユーザーはコピー&ペーストでYouTubeに貼り付けられる。
