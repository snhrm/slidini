# サムネイルプロンプト生成スキル

slidini プロジェクトの台本から YouTube サムネイル用の Gemini 画像生成プロンプトを作成する。

## プロンプトの出力形式

以下の構造で出力する。**曖昧な指示は避け、具体的なレイアウト・色コード・テキスト内容・サイズ比率まで指定する。**

```
【動画タイトル】
{動画のフルタイトル}

【動画の概要】
{動画の内容を3〜5行で要約。何が注目ポイントかを明確に}

上記の動画のサムネイルを作成してください。

アスペクト比: 16:9

【デザイン指示】
- 背景: {具体的な背景デザイン。ベースカラーのhexコード、背景に流れる演出の詳細}
- メインカラー: {アクセントカラーのhexコード、どこに使うか}
- レイアウト: {左右の配置、何をどこに置くか}
- テキスト要素:
  - {ラベル/バッジの詳細: 位置、背景色、文字色}
  - {メインキーワードの表示: フォントスタイル、サイズ感}
  - {数字やバッジなどの補足要素}
  - {日付や期間の表示}
- キャラクター配置: {添付画像のキャラクターの配置位置、ポーズの演出、動画内容との関連性、サイズ（画面の30〜40%程度）}
- 追加要素: {背景にさりげなく配置する装飾要素、動画テーマを示すアイコンやシンボル}
- 全体の雰囲気: {1〜2文で完成イメージを伝える}
```

## 入力の読み取り

台本ファイル（`*-script.md`）または `.slide.json` から以下を抽出:
- 動画タイトル
- カテゴリ
- 注目トピック・機能名（2〜3個、メインキーワードとして大きく表示する）
- 数字で表せるインパクト（「10バージョン更新」「5つの新機能」等）
- 対象期間・日付

## デザイン指示の書き方

### 背景
カテゴリに応じた具体的な背景を指示する。必ず hex カラーコードを含める。

| カテゴリ | 背景 | ベースカラー |
|---------|------|------------|
| Claude Code アップデート | ダークターミナル風、コードやコマンドラインが流れる演出 | `#0d1117` |
| 週刊IT技術ニュース | テック風グラデーション、回路パターン | `#0a1628` |
| 週刊AIニュース | ネイビー+光彩、ニューラルネットワーク風パターン | `#0f172a` |
| 週刊IT企業ニュース | ダークグリーン系、株価チャート風の線 | `#0a1f1a` |
| 技術解説・比較 | テーマに合った背景、関連アイコンを散りばめる | テーマ依存 |

### メインカラー（アクセント）
| カテゴリ | アクセントカラー |
|---------|---------------|
| Claude Code アップデート | オレンジ `#f97316` |
| 週刊IT技術ニュース | ティール `#06b6d4` |
| 週刊AIニュース | ブルー `#3b82f6` |
| 週刊IT企業ニュース | レッド/オレンジ `#ef4444` / `#f97316` |
| 技術解説・比較 | テーマ色 |

### レイアウト
- **左側**: テキスト情報（ラベル、メインキーワード、バッジ、日付）
- **右側**: ATIキャラクター（画面の30〜40%）

### テキスト要素（4層構造）
1. **ラベル/バッジ**（上部）: カテゴリ名。アクセントカラー背景 + 白文字
2. **メインキーワード**（中央）: 注目機能名やトピック名を2〜3個、大きく表示。モノスペースフォント風やテーマに合ったスタイル
3. **数字バッジ**（補足）: インパクトのある数字（「10バージョン更新」「満足度No.1」等）
4. **日付**（小さく）: 対象期間や公開日

### キャラクター配置
添付画像のATIキャラクターについて、以下を必ず指示する:
- 配置位置（右側）
- **動画内容に関連するポーズ・演出**（例: スマホを操作している、ターミナルを指し示している、虫眼鏡で調べている）
- サイズ目安: 画面の30〜40%

### 追加要素
背景にさりげなく配置して、動画テーマを視覚的に補強する要素:
- コマンドやコード片（ターミナル系）
- アイコンやロゴのシルエット（技術系）
- チャートやグラフの線（ニュース系）
- プロンプト記号 `>_` やキーボードショートカット表示（開発者向け）

## 出力

プロンプトテキストのみ出力する。ユーザーは `assets/img/character.png` と一緒に Gemini に貼り付けて画像を生成する。
